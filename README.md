# 🎁 Amigo Secreto – Challenger
Aplicação simples e interativa para gerenciar uma lista de participantes e realizar o sorteio do Amigo Secreto diretamente no navegador. Construída utilizando HTML, CSS e JavaScript puro.

# 🚀 Funcionalidades
- ➕ Adicionar nomes de participantes a uma lista
- 📜 Visualizar todos os participantes adicionados
- 🎯 Sortear aleatoriamente um participante como "Amigo Secreto"
- ⚡ Interface simples, responsiva e intuitiva

# 🛠️ Tecnologias utilizadas
- HTML5 para a estrutura da aplicação
- CSS3 para estilização e layout
- JavaScript para a lógica de adicionar nomes e realizar o sorteio
